How to Organize!

Cubestack Bot (@cubestackbot) identifies certain hash tags and uses them to organize. The source for the Bot is open source, and you can use it for any personal and Non commercial purpose. No registration required, just tweet to us with #List Hash code and we will take care of rest. Below are the Hash Tags it currently recognizes and responds, we will keep on adding new Hash updates.

  • List: Just reply to a Tweet and Quote it with #List adding @cubestackbot in the reply/mention, adds the quoted or replied tweet to list with the name. Example: This tweet adds the replied tweet to a list Named Awesome. A list name cannot have spaces and will be ignored by the Bot.

    @googlecloud @cubestackbot #LiSt AWESOME

    — Cube Stack (@cubestack) February 16, 2017
  • Help: Will tweet the details of the Hash Tags identified.

We are not case sensitive. Like any of replies to you by @cubestackbot and they will be deleted, in case you do not prefer history.
